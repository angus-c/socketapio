# An Event Driven API

This is super simple experiment, full article available here: http://teamstrobe.com/2014/02/10/sync-data-with-socketio-and-flight.html

## Requirements

* node.js with npm
* bower

## Setup

1. Clone the repo
2. run `npm install`
3. run `bower install`
4. run `node app.js`
5. visit `http://localhost:3000/`
